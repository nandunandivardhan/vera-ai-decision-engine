# VERA AI Decision Engine

This project is an AI-powered decision system built for the Magicpin VERA AI Challenge.

## Features
- Context-aware decision engine
- Personalized customer messaging
- Multi-category support (Restaurant, Salon, Grocery, Fashion)
- Behavioral strategy selection
- Dynamic offer generation
- Scoring system for quality evaluation

## Output
The system generates:
- Smart marketing messages
- Strategy decisions
- Confidence scores
- Business impact evaluation

## Tech
- Python
- Rule-based AI + heuristics

## How to Run
python compose_system_v2.py
